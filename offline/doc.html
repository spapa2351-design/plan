전략적 실행 로드맵 (Execution Plan)

안정적인 아키텍처 전환을 위해 3단계 타당성 검토 기반의 실행 로드맵입니다. 기존 시스템의 얽힌 레거시(Legacy)를 안전하게 회피하면서도 고객의 시급한 요구사항을 충족시키는 개발 순서를 제시합니다.

Phase 1

🏢 오프라인 코스 선(先) 개발

기존 온라인 DB를 건드리지 않고, '쉽고 편한 현대적 UI/UX'의 뼈대와 출결 내재화 모듈을 안전하게 테스트하고 확립합니다.

Game Changer
Phase 2

💻 온라인 코스 (가벼운 조립)

무거운 저작 도구를 완전히 탈피하고, 가벼운 '에셋 참조형 플레이리스트' 세팅을 구현하여 폭발적인 반응을 이끌어냅니다.

Phase 3

🔄 블렌디드 코스 완성

독립적인 모듈로 단단하게 완성된 오프라인 출석과 온라인 콘텐츠를 하나의 커리큘럼 UI 안에서 섞어주는(Mix) 융합 단계입니다.

영향도 경고: 공유 중인 에셋입니다.
